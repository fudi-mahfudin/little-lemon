# Little Lemon

The little lemon app is the capstone project for Meta's Front-End Developer Certificate on Coursera.

## 🚀 Technologies

This project was developed with the following technologies:
- React JS
- Typescript
- Vite
- Styled Components
- Styled Icons
- Formik
- Yup
- Vitest
- React Testing Library
